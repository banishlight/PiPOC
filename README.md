# PI POC 
By Cuba Giesbrecht

This project aims to create software for a raspberry pi to run inside of a car's center console taking the place of a stereo system, displaying OBD ECU data, and playing bluetooth audio.

## Build Process
Run the command 
```
make
``` 
it will compile the binary into the ```bin/``` directory and copy the files needed to install into that directory aswell.
